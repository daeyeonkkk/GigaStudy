# UI Mockup Exports

Date: 2026-04-09

This folder stores repo-visible mockup exports for the canonical GigaStudy screens.

Purpose:

- give implementation a concrete visual file to follow
- make design review possible inside the repo
- avoid relying only on prose wireframes for visual refactors

## Current Assets

- `home-v1.svg`
- `studio-v1.svg`
- `arrangement-v1.svg`
- `shared-review-v1.svg`
- `ops-v1.svg`

These are the seeded visual baseline exports for the canonical product screens.

They do not replace a shared editable Figma source.
They make the design target visible inside the repo until that editable source is fully established.

The current equivalent editable source lives in `../UI_EDITABLE_SOURCE/`.
